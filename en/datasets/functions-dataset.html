Functions

Collection of synthetic datasets generated applying continuous and limitated functions to a closed subset of domain; there are datasets done with scalar and vectorial functions and/or one-variable and multivariate functions. Datasets are in .csv format with header (each header column corresponds to a name of a variable) and in some case there is also the .arff format (for Weka); there are training/learning files and test files: first ones are bigger than seconds ones and the major part of data contained in test files are not present in training/learning ones.

Download of the dataset

The dataset is available on GitHub.
